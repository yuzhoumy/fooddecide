# CraveLogic

CraveLogic is a React web app that turns the PRD into an MVP food decision engine: taste fingerprint controls, situation dashboard, a single ranked recommendation, spin-again behavior, and a feedback loop stub ready for Supabase persistence.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The Vite config uses `/fooddecide/` as the production base so the app can be deployed on GitHub Pages for this repository.
